    **Create CartRepository domain interface:**
    Create `domain/src/main/java/com/wenubey/domain/repository/CartRepository.kt`:
    ```kotlin
    interface CartRepository {
        fun observeCartItems(userId: String): Flow<List<CartItem>>
        fun observeUniqueProductCount(userId: String): Flow<Int>
        suspend fun getCartItem(userId: String, productId: String): CartItem?
        suspend fun addToCart(userId: String, product: Product, quantity: Int)
        suspend fun updateQuantity(userId: String, productId: String, newQuantity: Int)
        suspend fun removeFromCart(userId: String, productId: String)
        suspend fun clearCart(userId: String)
        // Sync methods called by SyncWorker — write to Firestore
        suspend fun syncAddToCart(userId: String, productId: String, quantity: Int, snapshotPrice: Double)
        suspend fun syncUpdateQuantity(userId: String, productId: String, quantity: Int)
        suspend fun syncRemoveFromCart(userId: String, productId: String)
    }
    ```
    Import `com.wenubey.domain.model.CartItem`, `com.wenubey.domain.model.product.Product`, `kotlinx.coroutines.flow.Flow`.

    **Create CartRepositoryImpl:**
    Create `data/src/main/java/com/wenubey/data/repository/CartRepositoryImpl.kt`:
    - Constructor params: `CartItemDao`, `PendingOperationDao`, `ProductDao` (for stock validation), `SyncManager` (for SyncEvent emission), `FirebaseFirestore`, `DispatcherProvider`
    - `observeCartItems(userId)`: Return `cartItemDao.observeCartItems(userId).map { entities -> entities.map { it.toDomain() } }`
    - `observeUniqueProductCount(userId)`: Delegate to `cartItemDao.observeUniqueProductCount(userId)`
    - `getCartItem(userId, productId)`: Delegate to DAO, map to domain
    - `addToCart(userId, product, quantity)`:
      1. Get existing via `cartItemDao.getCartItem(userId, product.id)`
      2. If exists: call `cartItemDao.updateQuantity(userId, product.id, existing.quantity + quantity, Instant.now().toString())`
      3. If not exists: create CartItemEntity from product data with snapshotPrice = product.basePrice, availableStock = product.totalStockQuantity, and call `cartItemDao.upsert(entity)`
      4. Create PendingOperationEntity with type ADD_TO_CART (or UPDATE_CART_QUANTITY if existing), entityId = userId, payloadJson = serialized JSON with productId, quantity, snapshotPrice
      5. Insert into pendingOperationDao
      6. Call `SyncWorker.enqueue(context)` — get Application context from SyncManager or pass through constructor
      7. Emit `SyncEvent.OfflineWriteQueued` via SyncManager.syncEvents
    - `updateQuantity(userId, productId, newQuantity)`:
      1. If newQuantity <= 0: call `removeFromCart(userId, productId)` and return
      2. Call `cartItemDao.updateQuantity(userId, productId, newQuantity, Instant.now().toString())`
      3. Queue PendingOperationEntity with type UPDATE_CART_QUANTITY
      4. Enqueue SyncWorker
    - `removeFromCart(userId, productId)`:
      1. Call `cartItemDao.deleteItem(userId, productId)`
      2. Queue PendingOperationEntity with type REMOVE_FROM_CART
      3. Enqueue SyncWorker
    - `clearCart(userId)`: Call `cartItemDao.clearCart(userId)` — no queue (used on checkout success, Phase 4)
    - `syncAddToCart(userId, productId, quantity, snapshotPrice)`: Write to Firestore `users/{userId}/cart/{productId}` document with fields {productId, quantity, snapshotPrice, updatedAt}. Use `firestore.collection("users").document(userId).collection("cart").document(productId).set(data).await()`
    - `syncUpdateQuantity(userId, productId, quantity)`: Update Firestore document `users/{userId}/cart/{productId}` with {quantity, updatedAt}
    - `syncRemoveFromCart(userId, productId)`: Delete Firestore document `users/{userId}/cart/{productId}`

    Use `@Serializable` data classes for payloads: `AddToCartPayload(productId, quantity, snapshotPrice)`, `UpdateCartQuantityPayload(productId, quantity)`. Place them inside CartRepositoryImpl or in a `data/model/` package.
