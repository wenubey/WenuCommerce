    Create `app/src/main/java/com/wenubey/wenucommerce/customer/customer_cart/CartState.kt`:
    ```kotlin
    data class CartState(
        val cartItems: List<CartItem> = emptyList(),
        val isLoading: Boolean = true,
        val error: String? = null,
        val selectedItems: Set<String> = emptySet(), // productIds for bulk select
        val isSelectionMode: Boolean = false,
        val undoItem: CartItem? = null // for undo snackbar after removal
    )
    ```
    Derived properties (computed, not stored):
    - `subtotal: Double` = cartItems.filter { !it.isProductDeleted && it.availableStock > 0 }.sumOf { it.snapshotPrice * it.quantity }
    - `availableItemCount: Int` = cartItems.count { !it.isProductDeleted && it.availableStock > 0 }
    - `hasUnavailableItems: Boolean` = cartItems.any { it.isProductDeleted || it.availableStock <= 0 }
    - `canCheckout: Boolean` = cartItems.isNotEmpty() && cartItems.all { !it.isProductDeleted && it.availableStock > 0 }

    Create `app/src/main/java/com/wenubey/wenucommerce/customer/customer_cart/CartViewModel.kt` following the CustomerHomeViewModel pattern:
    - Constructor params: `CartRepository`, `AuthRepository`, `ProductDao` (for stock refresh), `DispatcherProvider`

      3. On each emission, cross-reference with ProductDao to refresh stock: for each cartItem, get ProductEntity from Room and update availableStock + isProductDeleted.       - UndoRemove: Call `cartRepository.addToCart(userId, product, item.quantity)` using the undoItem data
      - ToggleSelection / DeleteSelected / ToggleSelectionMode / ClearSelection: Manage selectedItems set in state for bulk delete feature
      - DeleteSelected: For each productId in selectedItems, call removeFromCart. Clear selection after.
