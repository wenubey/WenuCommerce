        data class UndoRemove(val item: WishlistItem) : WishlistAction
        data class AddItemToCart(val item: WishlistItem) : WishlistAction
        data object AddAllToCart : WishlistAction
        data object AddSelectedToCart : WishlistAction
        data class ToggleSelection(val productId: String) : WishlistAction

        data class NavigateToProduct(val productId: String) : WishlistAction

    Create `app/src/main/java/com/wenubey/wenucommerce/customer/customer_wishlist/WishlistViewModel.kt` following the CartViewModel pattern:
    - Constructor params: `WishlistRepository`, `CartRepository`, `AuthRepository`, `ProductDao` (for stock refresh), `DispatcherProvider`

    - `_state = MutableStateFlow(WishlistState())`

    - On each emission, cross-reference with ProductDao to refresh stock/deleted status (same pattern as CartViewModel)
    - `onAction(action: WishlistAction)`:

      - **AddAllToCart:** For each available item (not deleted, stock > 0), call addToCart. Skip unavailable. Show snackbar with count added.
      - **AddSelectedToCart:** Same as AddAllToCart but only for items in selectedItems set
      - **ToggleSelection/ClearSelection:** Manage selectedItems set
      - **NavigateToProduct/NavigateToHome:** These are UI events — emit via a SharedFlow or callback

    Create a reusable `WishlistHeartButton` composable (can live in `core/components/` or inline in the files that use it). This composable:
    - Parameters: `isWishlisted: Boolean`, `onToggle: () -> Unit`, `modifier: Modifier`
    - Uses `Animatable` + `LaunchedEffect` for scale bounce on toggle (per research Pattern 7):
      ```kotlin
      val scale = remember { Animatable(1f) }
      LaunchedEffect(isWishlisted) {
          scale.animateTo(1.3f, spring(dampingRatio = Spring.DampingRatioLowBouncy))
          scale.animateTo(1f, spring(dampingRatio = Spring.DampingRatioMediumBouncy))
      }
      ```
    - Icon: `Icons.Filled.Favorite` (filled red) when wishlisted, `Icons.Outlined.FavoriteBorder` (surface color) when not
    - Applied via `Modifier.graphicsLayer { scaleX = scale.value; scaleY = scale.value }`
