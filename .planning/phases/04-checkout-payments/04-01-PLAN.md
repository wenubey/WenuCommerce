<task type="auto">
  <name>Task 2: Implement createPaymentIntent Cloud Function</name>
  <files>
    functions/src/index.ts
  </files>
  <action>
    Create `functions/src/index.ts` with the `createPaymentIntent` callable function:

    **Imports:** `onCall`, `HttpsError` from `firebase-functions/v2/https`; `defineSecret` from `firebase-functions/params`; `Stripe` default import; `firebase-admin` with `initializeApp()`.

    **Initialize Firebase Admin:** `admin.initializeApp()` at module level.

    **Define secret:** `const stripeSecretKey = defineSecret("STRIPE_SECRET_KEY")`

    **Function signature:** `export const createPaymentIntent = onCall({ secrets: [stripeSecretKey] }, async (request) => { ... })`

    **Function body:**
    1. **Auth check:** If `!request.auth`, throw `new HttpsError("unauthenticated", "Must be signed in to checkout")`
    2. **Extract data:** `const { cartItems, shippingAddress, userId } = request.data` — type cartItems as `Array<{ productId: string; productTitle: string; quantity: number; price: number }>`
    3. **Input validation:** Verify cartItems is non-empty array, each item has productId/quantity/price; throw `HttpsError("invalid-argument", ...)` if invalid
    4. **Server-side stock validation:** For each cartItem, read `admin.firestore().collection("products").doc(item.productId).get()`. Check `productDoc.data()?.totalStockQuantity >= item.quantity`. If any fail, throw `HttpsError("failed-precondition", "Insufficient stock for {productTitle}")` with details of which items failed.
    5. **Compute subtotal in cents:** `cartItems.reduce((sum, item) => sum + Math.round(item.price * 100) * item.quantity, 0)`
    6. **Compute shipping in cents:** For each cart item, read product doc's `shipping.shippingCost` (already fetched above). Shipping is per-product-line (not per-unit): `Math.round(shippingCost * 100)` for each unique product line. Sum all shipping costs.
    7. **Total:** `subtotalCents + shippingCents`
    8. **Create Stripe PaymentIntent:**
       ```
       const stripe = new Stripe(stripeSecretKey.value(), { apiVersion: "2025-01-27.acacia" })
       const paymentIntent = await stripe.paymentIntents.create({
         amount: totalCents,
         currency: "usd",
         automatic_payment_methods: { enabled: true },
         metadata: { userId: request.auth.uid, orderId }
       })
       ```
    9. **Create pending Order in Firestore:** Generate orderId via `admin.firestore().collection("orders").doc().id`. Write:
       ```
       {
         userId: request.auth.uid,
         status: "PENDING",
         subtotal: subtotalCents / 100,
         shippingTotal: shippingCents / 100,
         totalAmount: totalCents / 100,
         currency: "USD",
         stripePaymentIntentId: paymentIntent.id,
         shippingAddress: shippingAddress,
         items: cartItems.map(item => ({
           productId: item.productId,
           productTitle: item.productTitle,
           quantity: item.quantity,
           snapshotPrice: item.price,
           lineTotal: item.price * item.quantity
         })),
         createdAt: admin.firestore.FieldValue.serverTimestamp(),
         updatedAt: admin.firestore.FieldValue.serverTimestamp()
       }
       ```
    10. **Return:** `{ clientSecret: paymentIntent.client_secret, amountCents: totalCents, orderId }`

    **Important:** Shipping is per-product-line, not per-unit. If a customer buys 3 units of Product X, shipping cost applies once for that line item per the research open question resolution.

    Compile the function: `cd functions && npx tsc`
  </action>
