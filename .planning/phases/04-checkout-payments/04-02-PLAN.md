    **AddressRepositoryImpl.kt:**
    - Constructor: `(private val addressDao: AddressDao, private val firestore: FirebaseFirestore)`
    - Addresses stored as a subcollection: `users/{userId}/addresses/{addressId}` in Firestore.

    - **Room-first pattern** (same as CartRepository, WishlistRepository per locked user decision: "Addresses are saved to user profile (Firestore + Room) for reuse across checkouts"):

    - `observeSavedAddresses(userId)`:
      1. Return `addressDao.observeByUser(userId).map { entities -> entities.map { it.toDomain() } }`
      2. In `init {}` or via a `startSync(userId)` method, set up a Firestore snapshot listener on `users/{userId}/addresses` that writes through to Room:
         ```kotlin
         firestore.collection("users").document(userId)
             .collection("addresses")
             .addSnapshotListener { snapshot, error ->
                 if (error != null || snapshot == null) return@addSnapshotListener
                 coroutineScope.launch {
                     val entities = snapshot.documents.mapNotNull { doc ->
                         // parse doc to AddressEntity
                     }
                     addressDao.deleteAllForUser(userId)
                     addressDao.upsertAll(entities)
                 }
             }
         ```
      3. This ensures Room always has the latest data, and the Flow from Room emits automatically.

    - `saveAddress(userId, address)`:
      1. Generate addressId if `address.id` is empty: `UUID.randomUUID().toString()`
      2. Write to Firestore: `firestore.collection("users").document(userId).collection("addresses").document(addressId).set(address.toMap()).await()`
      3. Write to Room: `addressDao.upsert(address.copy(id = addressId).toEntity(userId))`
      4. The Firestore snapshot listener will also pick this up, but writing to Room immediately ensures instant local availability.

    - `deleteAddress(userId, addressId)`:
      1. Delete from Firestore: `firestore.collection("users").document(userId).collection("addresses").document(addressId).delete().await()`
      2. Delete from Room: `addressDao.delete(userId, addressId)`
